## UIコンポーネント一覧

### 1. ボタンコンポーネント

#### Primary Button
```
用途: メインアクション（トレーニング開始、STARTなど）
サイズ: 横幅90%、高さ56px
スタイル: 角丸8px、グラデーション背景
状態: default / hover / pressed / disabled
```

#### Secondary Button
```
用途: サブアクション（記録、キャンセルなど）
サイズ: 横幅90%、高さ48px
スタイル: 角丸8px、ボーダーのみ
状態: default / hover / pressed
```

#### Icon Button
```
用途: ヘッダーのメニュー、戻るボタン
サイズ: 44x44px（タップ領域）
スタイル: アイコンのみ、背景透明
```

### 2. カードコンポーネント

#### Song Card
```
用途: 曲選択画面の曲情報表示
構成: サムネイル(80x80) + メタ情報 + 選択状態
スタイル: 背景白、影付き、角丸12px
インタラクション: タップで選択、選択時は枠強調
```

#### Record Card
```
用途: トレーニング記録の詳細表示
構成: 曲名 + 種目 + スコア + 経験値 + 時刻
スタイル: 背景グレー、角丸8px
```

### 3. プログレスバー

#### Experience Bar
```
用途: 経験値の進捗表示
サイズ: 横幅80%、高さ24px
スタイル: 角丸12px、グラデーション塗り
アニメーション: 増加時にスムーズに伸びる
```

#### Judgment Bar
```
用途: リザルト画面の判定内訳
サイズ: 可変幅、高さ16px
スタイル: 角丸4px、判定種別で色分け
```

### 4. キャラクター表示

#### Character Sprite
```
用途: ホーム画面のキャラクター
サイズ: 200x200px
アニメーション:
  - 待機: ゆっくり上下・左右に揺れる (2秒ループ)
  - 進化: 光のエフェクト + 拡大縮小 (3秒)
  - レベルアップ: 跳ねる動作 (1秒)
```

### 5. カレンダーコンポーネント

#### Monthly Calendar
```
用途: トレーニング記録の月間表示
構成: ヘッダー(年月) + 曜日行 + 日付グリッド(7x6)
マーカー: トレーニング日に●、本日に⭕️
インタラクション: 日付タップで詳細表示
```

### 6. ゲームプレイUI

#### Note Object
```
用途: 降下するノーツ
サイズ: 60x60px（円形）
スタイル: グロー効果、レーン別で色分け
アニメーション: 上から下へ一定速度で移動
```

#### Target Line
```
用途: タップ判定位置
スタイル: 太さ4px、明るい色、パルスアニメーション
位置: 画面下部から120px
```

#### Judgment Feedback
```
用途: タップ判定結果の表示
種類: Perfect(金) / Great(青) / Good(緑) / Miss(赤)
表示: 0.5秒フェードイン→フェードアウト
位置: ターゲットライン上
```

## インタラクション仕様

### 1. ホーム画面

#### キャラクターアニメーション
```
待機モーション:
  - 2秒周期で上下に5px揺れる
  - 同時に左右に3度回転
  - イージング: ease-in-out

タップ時:
  - 0.1秒で10%縮小
  - 0.2秒で元のサイズに戻る
  - ランダムな効果音再生
```

#### ボタン操作
```
タップ時:
  - 0.1秒で95%に縮小
  - リップルエフェクト（波紋）
  - ハプティックフィードバック（軽い振動）
  - 0.3秒後に画面遷移
```

### 2. 曲選択画面

#### 難易度切替
```
タブタップ:
  - 0.2秒でスライドアニメーション
  - 曲リストを0.3秒でフェードイン
  - アクティブタブに下線表示
```

#### 曲カード選択
```
タップ時:
  - 枠線を太く（2px → 4px）
  - 背景色を明るく
  - チェックマーク表示
  - 選択音再生

長押し:
  - プレビュー音声再生（5秒）
  - カードを5%拡大
```

### 3. ゲーム準備画面

#### START ボタン
```
タップ時:
  - カウントダウン表示（3, 2, 1, GO!）
  - 各数字0.8秒表示、0.5秒でフェードアウト
  - GO!と同時にゲーム開始
  - 開始音再生
```

### 4. ゲームプレイ画面

#### ノーツ判定
```
Perfect (±30ms):
  - 金色のエフェクト爆発
  - +500点
  - "Perfect!" テキスト表示
  - 強いハプティック

Great (±60ms):
  - 青色のエフェクト
  - +300点
  - "Great!" テキスト表示
  - 中程度ハプティック

Good (±100ms):
  - 緑色のエフェクト
  - +100点
  - "Good" テキスト表示
  - 弱いハプティック

Miss (100ms超 or 未タップ):
  - 赤色の×マーク
  - 0点
  - "Miss..." テキスト表示
  - コンボリセット
```

#### コンボシステム
```
10コンボ毎:
  - フォントサイズ拡大（1.2倍）
  - スコア倍率アップ（x1.1, x1.2...）
  - 「10 Combo!」などのテキスト表示
```

#### 背景アニメーション
```
運動フォーム:
  - 曲のBPMに合わせてループ再生
  - 透明度50%
  - スムーズな動作（60fps目標）
```

### 5. リザルト画面

#### 登場アニメーション
```
1. スコアが0から最終値まで1秒でカウントアップ
2. 判定内訳バーが0.5秒で左から伸びる（順次）
3. 経験値が0から獲得量まで1秒でカウントアップ
4. 経験値バーが0.5秒で伸びる
5. 進化判定（達成時は進化演出へ）
```

#### 進化演出（経験値が閾値到達時）
```
1. 画面全体が白くフラッシュ（0.3秒）
2. キャラクターが光に包まれる（1秒）
3. 次の段階のキャラクターに変化（0.5秒）
4. 「進化しました！」テキスト表示
5. 紙吹雪エフェクト（2秒）
6. 進化音楽再生
```

### 6. トレーニング記録画面

#### カレンダー操作
```
日付タップ:
  - 0.2秒で選択状態に変化
  - 下部に詳細パネルがスライドイン（0.3秒）
  - トレーニングデータを表示

月切替:
  - 左右スワイプで月移動
  - 矢印ボタンでも切替可能
  - 0.4秒でスライドアニメーション
```

#### スクロール
```
詳細パネル:
  - 縦スクロール可能
  - 慣性スクロール有効
  - オーバースクロール時にバウンス効果
```

### 7. 共通インタラクション

#### 画面遷移
```
進む（前方）:
  - 新画面が右からスライドイン（0.3秒）
  - 現画面が左へスライドアウト

戻る（後方）:
  - 現画面が右へスライドアウト（0.3秒）
  - 前画面が左からスライドイン
```

#### ローディング
```
表示:
  - スピナー（円形、回転）
  - 「読み込み中...」テキスト
  - 背景半透明オーバーレイ

データ取得時:
  - 最小表示時間: 0.5秒（チラつき防止）
  - 3秒超過時: タイムアウトエラー表示
```

#### エラー表示
```
トースト通知:
  - 画面下部から0.3秒でスライドアップ
  - 3秒間表示後、自動で消える
  - タップで即座に消去可能
  - エラー音再生（オプション）
```

#### ハプティックフィードバック
```
軽い: ボタンタップ、選択
中程度: 成功アクション、Great判定
強い: 重要なアクション、Perfect判定、進化
警告: エラー、Miss判定
```
